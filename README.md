# RemotePlayer 🎵

Удалённое управление VK Музыкой с телефона.

## Возможности
- 🎧 Воспроизведение музыки из VK
- 📱 Управление с Android-телефона по Wi-Fi
- 🔀 Shuffle и автовоспроизведение
- 🔊 Регулировка громкости
- 💾 Кэширование треков (офлайн-режим)
- 🚀 Потребляет ~40 МБ RAM

## Стек
- **Сервер:** ASP.NET Core 8, VkNet, NAudio, FFmpeg
- **Клиент:** Kotlin, Jetpack Compose, Retrofit, Koin
